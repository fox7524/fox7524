# Hi there, I'm Kayra (fox) Yılmaz 👋

[![GitHub Streak](https://streak-stats.demolab.com/?user=fox7524&theme=dark)](https://git.io/streak-stats) 

**Junior Maker | Robotics & Aerospace Developer | Cinematographer**

I am an 11th-grade student at Ahmet Keleşoğlu Fen Lisesi in Istanbul, building the bridge between hardware engineering, software development, and digital media. With my sights set on Electrical-Electronics or Mechanical Engineering, my work spans from programming ESP32 (specifically N8R2 models) and training local LLMs, to designing fixed-wing UAVs and directing award-winning short films. 

Currently, I'm developing **Ducofex Industries**, a startup focused on home-sized CNC machines, and leading the **Üsküdar UAV Team**.
